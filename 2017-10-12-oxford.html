Day 2 Setup

1) Make new repository on github

1a) From webpage for new repositoty, green button Clone or Download - copy the URL

2) Open RStudio

File -> New Project - > Version Control -> Git

Paste the URL that you just copied

Create Project

RStudio not finding Git on Mac OS? 

Change the Git executable path in RStudio > Preferences > Git/Svn from /usr/bin/git to /usr/local/bin/git and then restart RStudio.

calcGDP <- function(dat, year=NULL, country=NULL) {
  if(!is.null(year)) {
    dat <- dat[dat$year %in% year, ]
  }
  if (!is.null(country)) {
    dat <- dat[dat$country %in% country,]
  }
  gdp <- dat$pop * dat$gdpPercap
  
  new <- cbind(dat, gdp=gdp)
  return(new)
}
